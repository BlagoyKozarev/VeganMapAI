# 📞 Как да намерите Replit Support

## Официални канали за поддръжка:

### 1. Support Форма (Най-добър вариант)
**https://replit.com/support**
- Кликнете "Contact Support" 
- Изберете "Technical Issue"
- Опишете проблема с production базата

### 2. Ask Forum (Бърза помощ от общността)
**https://ask.replit.com**
- Постнете въпроса си
- Общността и staff отговарят бързо
- Търсете категория "Deployments"

### 3. Discord сървър
**https://discord.gg/replit**
- Много активна общност
- Канал #help за въпроси
- Replit staff също помагат

### 4. Twitter/X
**@replit**
- За спешни проблеми
- Публични съобщения получават бърз отговор

## Какво да напишете:

### На английски:
```
Subject: Production database empty after deployment

My VeganMapAI deployment has an empty production database.
- Development DB: 408 restaurants ✓
- Production DB: 0 restaurants ✗
- Deployment URL: vegan-map-ai-bkozarev.replit.app

I cannot find the production DATABASE_URL in deployment settings.
Please help import data to production.

Thank you!
```

### На български (ще преведат):
```
Тема: Празна production база след deployment

Моят VeganMapAI deployment има празна production база данни.
- Development база: 408 ресторанта ✓
- Production база: 0 ресторанта ✗
- Deployment URL: vegan-map-ai-bkozarev.replit.app

Не мога да намеря production DATABASE_URL в настройките.
Моля помогнете с импорта на данните.

Благодаря!
```

## Очаквано време за отговор:
- Support форма: 24-48 часа
- Ask Forum: 2-6 часа
- Discord: 15-60 минути

## Съвет:
Използвайте Ask Forum или Discord за най-бърз отговор!
