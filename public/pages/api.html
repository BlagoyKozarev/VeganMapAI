VeganMapAI API

Early Access

Get programmatic access to our comprehensive vegan restaurant database and AI scoring system. Perfect for developers building vegan-focused applications, research projects, or integrations.

What's Included

  • Access to 477+ vegan-friendly restaurants in Sofia
  • Real-time AI vegan scoring (0-10 scale)
  • Detailed score component breakdowns
  • Location-based search and filtering
  • Cuisine types and price levels
  • RESTful JSON API with simple authentication
  • Rate limit: 1000 requests/day (beta)
