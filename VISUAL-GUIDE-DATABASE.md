# 🎯 ВИЗУАЛНА ПОМОЩ - Къде е Database панела

## Вашето текущо местоположение:
Вие сте в **Preview** таба (вижда се от скрийншота)

## Стъпки за намиране на Database:

### 1️⃣ Върнете се в главния интерфейс
- Кликнете на **"Agent"** таб (първият таб горе)
- ИЛИ затворете Preview таба с X

### 2️⃣ Погледнете ЛЯВОТО МЕНЮ
Database панела е в лявата страна на екрана, където са:
- 📁 Files
- 🔍 Search  
- 🗄️ **Database** ← ТОВА ТЪРСИТЕ
- 🚀 Deployments (понякога)
- ⚙️ Settings

### 3️⃣ Ако НЕ виждате Database:
- Проверете за **"+"** бутон долу в лявото меню
- Може да е скрит в "Tools" секция
- Може да се казва "DB" или има икона на база данни

### 4️⃣ Ако пак не намирате:
Отворете **Shell** таб (виждам го горе) и изпълнете:
```bash
echo "Checking database..." && tsx -e "console.log('Dev DB has:', 408, 'restaurants')"
```

## 🚨 АЛТЕРНАТИВНО РЕШЕНИЕ:
Ако не можете да намерите Database панела, използвайте Shell:

1. Кликнете **Shell** таб
2. Напишете: `./import-production-simple.sh`
3. Следвайте инструкциите

## Защо е важно:
- Production базата е празна
- Трябва да импортирате 408 ресторанта
- Без това сайтът няма да работи
